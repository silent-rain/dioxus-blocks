# Dioxus Blocks Components

```text
project/
├─ assets/ # 应用程序使用的资源文件目录
    ├─ css/ # SCSS/CSS 格式的样式文件
    ├─ img/ # 图片文件
├─ src/ # 源代码目录
│  ├─ lib.rs # 库入口文件，导出所有公共组件和常量
│  ├─ style.rs # 样式相关模块，提供样式辅助函数和常量
│  ├─ constant.rs # 常量定义，包含 CSS 资源引用
│  ├─ components # 组件实现目录
│  │  ├─ button.rs # Button 组件实现，提供可自定义的按钮
│  │  └─ card.rs # Card 组件实现，提供卡片容器和内容布局
├─ Cargo.toml # 项目配置文件，定义依赖项和特性标志
```
